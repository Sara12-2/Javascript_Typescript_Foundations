# 📘 JavaScript & TypeScript Cheat Sheets

A beginner-friendly collection of JavaScript and TypeScript cheat sheets designed to help developers learn core concepts, revise important topics, and prepare for technical interviews.

Whether you're starting your web development journey or refreshing your knowledge before an interview, these cheat sheets provide a quick and practical reference to the most commonly used concepts.

---

## 🚀 Why Learn JavaScript?

JavaScript is the programming language of the web. It allows developers to create interactive and dynamic websites by handling user interactions, manipulating web pages, processing data, and communicating with servers.

With JavaScript, you can build:

* Interactive Websites
* Single Page Applications (SPAs)
* Full-Stack Applications
* Mobile Applications
* Desktop Applications
* Server-Side Applications using Node.js

### Topics Covered

* Variables & Data Types
* Arrays & Objects
* Operators
* Functions
* Loops & Conditional Statements
* Strings & String Methods
* Regular Expressions
* Numbers & Math Methods
* Date Handling
* DOM Manipulation
* Browser APIs
* Events & Event Handling
* Error Handling

---

## 🔷 Why Learn TypeScript?

TypeScript is a superset of JavaScript that adds static typing and powerful development features. It helps developers write safer, cleaner, and more maintainable code by catching errors during development instead of runtime.

TypeScript is widely used in modern frameworks such as:

* React
* Next.js
* Angular
* NestJS

### Topics Covered

* Basic Types
* Any & Void
* Objects
* Template Literals
* Interfaces
* Union Types
* Functions
* Optional Parameters
* Rest Parameters
* Arrow Functions
* Classes
* Access Modifiers
* Readonly Properties
* Static Members
